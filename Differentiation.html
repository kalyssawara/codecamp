Differentiation

Tag Similarity Difference
hr & br Tag is an empty tag. It has no end tag. The hr tag is used to separate content but the br tag just insert a single line break
ul & ol Use the li tag to create lists The ul tag defines a bulleted list but the ol tag can be numerical
select & checkbox & radiobutton Display a list of options which a user can select form. Select is used to creat drop-down list, checkbox allows user to select one or more options but radiobutton will allow only one option.
div & span They aer used in conjunction with CSS div element is block-line and used to group larger chunks of code but span element is in-line and used for a small chunk of HTML inside a line.
script & style They are used to embed or reference executable code. The script tag is used to define a client-side script but the style tag is used to define style information of HTML.
